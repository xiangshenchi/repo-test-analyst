## [1.0.0] - 2025-05-13

### 新增 / Added
- 首次发布 `repo-test-analyst` Claude Skill
- 五阶段工作流：探索 → 识别 → 制定策略 → 分析 → 生成报告
- 语言支持：Python、JavaScript、TypeScript、Java、Go
- 静态分析集成：pylint、radon、bandit、ESLint、tsc
- 依赖审计：pip-audit、npm audit
- 六种报告类型：综合摘要、代码质量、安全漏洞、依赖风险、API 接口、测试用例
- 各语言测试策略参考文档
- 完整的 Markdown 报告模板
- OWASP Top 10 安全检查清单
- 基于 1–10 分的多维度健康度评分体系
